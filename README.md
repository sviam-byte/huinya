# Time Series Connectivity Toolkit

Выполняет:
1. Анализ связности (Connectivity)
Линейные методы: Полная и частная корреляция (Partial Correlation).
Частотные методы: Когерентность (Coherence) для поиска синхронизации на определенных частотах.
Нелинейные методы: Взаимная информация (Mutual Information) на базе KNN-оценок.
2. Поиск причинно-следственных связей (Causality)
Granger Causality: Проверка направленного влияния через векторную авторегрессию (VAR).
Transfer Entropy: Оценка направленного потока информации между рядами (библиотека pyinform).
Non-linear AH Matrix: Оценка нелинейной направленной связности.
3. Анализ внутренней динамики
Показатель Херста (Hurst Exponent): Оценка персистентности (памяти) ряда методами R/S, DFA и Wavelet.
Энтропия (Sample Entropy): Количественная оценка регулярности и хаотичности данных.
Спектральный анализ (FFT): Автоматическое построение АЧХ/ФЧХ, поиск доминирующих периодов и анализ сезонности.
4. Автоматизация отчетности
Excel Export: Генерация многостраничного отчета (20+ листов), включающего:
Тепловые карты (Heatmaps) для всех типов связей.
Коннектомы: Визуализация структуры связей в виде графов (NetworkX).
Результаты проверки на стационарность (ADF-тест) и поиск выбросов.
Автоматическая цветовая маркировка стат-значимых связей.

## Ограничения корректности (важно знать)

- Данные должны быть синхронны по частоте дискретизации (равный шаг по времени).
- Для лаговых методов нужно достаточно точек: рекомендуется минимум 10–20 точек на лаг.
- Для устойчивых оценок на практике требуется хотя бы 200–300 наблюдений на ряд.
